This is a prototype of a Payroll application implemented in Java. 
The application uses a local mysql database, and can be executed by opening application.jar in the application folder.
The application lets you view the total list of the employees.
Then a summary view of employees by department with the corresponding salaries can also be viewed.
Print Pay Slip is a menu that lets you print the Pay Slip for a selected employee.
An employee can be added to the database
The departments with descriptions can also be added, updated or deleted.
The position an employee can take can also be added, updated or deleted.
Under the folder sourcecode the code for the application is accessible.
